자바스크립트란?

웹 브라우저에서 사용하는 프로그래밍 언어입니다.
웹페이지, 웹 서버, 게임, 데스크톱/모바일 애플리케이션 개발 데이터베이스 관리 등 많은 것을 할 수 있습니다.

자바스크립트 활용 분야

  • 웹 클라이언트 애플리케이션 개발 -웹 문서 작성 도구 (구글 문서, 마이크로소프트 오피스 온라인, 크롬 웹 스토어 등)
  • 웹 서버 애플리케이션 개발 -Node.js

    장점

  • 웹 서버 애플리케이션을 개발할 때 꼭 필요한 간단한 모듈만을 제공
  • 빠른 속도
  • 모바일 애플리케이션 개발 -모든 스마트폰에서 인식할 수 있는 공통된 프로그래밍 언어이기 떄문에
    제작 비용을 줄일 수 있습니다. ex)페이스북, 인스타, 핀터레스트, 우버 등
  • 데스크톱 애플리케이션 개발데이터베이스 관리 -VS Code, 디스코드,깃허브, 워드프레스 등
  • 데이터베이스 관리 -NoSQL, MongoDB

기초 용어

  • 표현식: 값을 만들어내는 간단한 코드
  • 문장: 하나 이상의 표현식이 모여 구성, 코드를 읽어들이는 기본 단위
  • 키워드: 프로그래밍 언어가 처음 만들어질 때 정해진 특별한 의미가 있는 단어
  • 식별자: 이름을 붙일 때 사용하는 단어
  • 주석: 프로그램 코드를 설명하는 문장, 프로그램 진행에는 영향을 주지 않음.
  • 자료: 프로그래밍에서 프로그램이 처리할 수 있는 모든 것
  • 자료형: 자바스크립트에서는 숫자, 문자열, 불 자료형의 형태로 나눔.
  • 상수: 변하지 않는 값을 지정하는 식별자. const 키워드를 사용해 선언
  • 변수: 변하는 값을 지정하는 식별자. let 키워드를 사용해 선언
  • 선언: 상수 또는 변수를 생성
  • 할당: 상수 또는 변수에 값을 넣는 것.
